## 説明文（4000文字以内）
```
衝動が来た瞬間に開くアプリです。

神経科学の研究によると、感情の波のピークは約90秒で収まります。食欲・SNS・買い物・タバコ・アルコール・夜更かし・怒りなど、あらゆる衝動に対して「ただ観察する」ことで、自然に波が引いていきます。

【主な機能】

▶ 90秒サーフィン
衝動が来たらタップするだけ。科学的なアドバイスが流れ、波が引くのをサポートします。

▶ 14種類の衝動タイプ
食べ過ぎ・間食・SNS・買い物・夜更かし・ポルノ・タバコ・アルコール・怒り・ネガティブ思考・感情食い・承認欲求・先延ばし・その他。あなたの課題に合わせたアドバイスが表示されます。

▶ 実績バッジシステム
21種類のバッジで継続をサポート。「最初の一歩」から「百の波」まで、積み重ねが目に見える形で残ります。

▶ サウンド（雨・波・森・アンビエント）
観察中は4種類の自然音で集中をサポート。音量調整も可能。

▶ 未来設計
なりたい自分のビジョンと、衝動が来たときの代替行動をあらかじめ設定。90秒が終わったときに表示され、本当の目標を思い出せます。

▶ 完全なプライバシー
全データはあなたのデバイスにのみ保存。外部サーバーへの送信はゼロ。インターネット接続不要。

【こんな方におすすめ】
・食欲やお菓子への衝動をコントロールしたい方
・SNSや動画を開く習慣を減らしたい方
・衝動買いを減らしたい方
・禁煙・禁酒に取り組んでいる方
・怒りや感情の波を落ち着かせたい方
・先延ばし癖を改善したい方

完璧にやめる必要はありません。衝動が来るたびに90秒だけ観察してみてください。それだけで、少しずつ変わっていきます。
```

## 審査メモ（Review Notes）
```
このアプリはCBT（認知行動療法）とACT（アクセプタンス＆コミットメントセラピー）の原則に基づいた衝動制御アプリです。

【データについて】
全てのデータはデバイスのlocalStorageにのみ保存されます。外部サーバーへの通信は一切ありません。ネットワーク権限は使用していません。

【ログインについて】
アカウント登録・ログインは不要です。アプリを開くとすぐに使用できます。

【審査用テスト手順】
1. アプリを起動
2. オンボーディングで目標（例：「暴食を減らしたい」）を選択
3. 理由・声のタイプを入力して「はじめる」をタップ
4. ホーム画面の「衝動がきた」ボタンをタップ
5. 90秒間のサーフィン画面が表示される
6. 設定タブからデータエクスポートや各種設定が可能
```
